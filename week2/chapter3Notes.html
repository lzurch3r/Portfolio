Ch. 3 Arrays, Logic, and Loops

Arrays

Arrays are not primitive values but a special built-in object.

Arrays can be created empty, or they can be initialized with data.

Adding values to an array will automatically increase the size of that array. Arrays can hold different types of items. Deleting items from an array using the keyword delete will remove that item and replace it with undefined. The space that the deleted item occupied will persist.

Destructuring an array means taking values out of an array and presenting them as individual values:

const [x,y] = [1,2];

x << 1

y << 2

Destructuring can swap the value of two variables over:

[x,y] = [y,x];

x << 2

y << 1

Arrays use pop(), push(), shift(), and unshift() method. Pop and Push are used to remove items from and add items to the end of an array. Shift and Unshift are used to remove items from and add items to the beginning of an array.

You can use the concat() method to join arrays together, or you can use the new spread operator:

avengers = [ ...avengers, ...['Hulk','Hawkeye', 'Black Widow'] ];

<< ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow']

The slice(start, finish) method finds the items in the array starting at the specified starting point and ending before the finishing point. The splice(index, numRemove, newItem) method removes items from an array and inserts new items in their place.

Use the indexOf() method to find the index of an item in an array. If the method returns the value -1, it is not in the array.

Use the includes() method to return a boolean value determining whether the array contains a particular element or not. You can add an extra parameter to indicate which index to start the search from.

Sets

Sets are basically arrays that cannot contain duplicate values. They can also only be created using:

const list = new Set();

Numbers can be added during creation:

const numbers = new Set([1,2,3]);

NOTE: All non-primitive values, such as arrays and objects, are considered unique values, even if they contain the same values. also, type coercion is not used when values are added to a set.

The has() method checks for a value in a set, returning a boolean value (true or false). The delete() method removes a value from a set, and clear() removes all values from a set.

Sets can be converted to arrays. First, create a set (the amount of items is arbitrary), then convert it:

const shoppingArray = [...shoppingSet];

You can also use the Array.from() method:

const shoppingArray = Array,from(shoppingSet);

Maps

Maps are a useful way to contain key-value pairs:

romanNumerals.set(1, 'I');

<< Map { 1 => 'I' }

Multiple items can be added by repeatdely calling the set() method. Also, a map can be created with multiple values by using a nested array as a parameter:

const heroes = new Map([ ['Clark Kent','Superman'], ['Bruce Wayne','Batman'] ]);

Maps can be converted into a nested array of key-value pairs:

[...romanNumerals]

<< [ [ 1, 'I' ], [ 2, 'II' ], [ 3, 'III' ], [ 4, 'IV' ], [ 5, 'V' ] ]

OR

Array.from(romanNumerals)

<< [ [ 1, 'I' ], [ 2, 'II' ], [ 3, 'III' ], [ 4, 'IV' ], [ 5, 'V' ] ]

Loops

The different types of loops are: while, do ... while, and for loops.

NOTE: Arrays and sets can be iterated through with for-of loops.
